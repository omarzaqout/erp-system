# 🚀 نظام Antigravity ERP المتكامل | Antigravity ERP System

نظام تخطيط موارد المؤسسات (ERP) متكامل، مبني باستخدام تقنيات حديثة ليجمع بين مرونة **Odoo** وقوة الضوابط المالية في **SAP Business One**. تم تصميمه ليكون حلاً احترافياً للمؤسسات المتوسطة والكبيرة.

---

## 🏗️ موديولات النظام الحالية (Core Modules)

### 🏎️ 1. نقطة البيع المتقدمة (Advanced POS)
*   **إدارة الورديات (Shift Management):** فتح وإغلاق وردية مع رصد العهدة النقدية.
*   **دعم الباركود:** كفاءة عالية في إضافة المنتجات باستخدام الماسح الضوئي.
*   **تعدد العملات اللحظي:** البيع بمختلف العملات (USD, IQD, ILS) مع تحويل آلي للمحاسبة.
*   **الامتثال الضريبي (ZATCA):** توليد فواتير إلكترونية متوافقة مع متطلبات هيئة الزكاة والدخل (QR Code).
*   **المرتجعات والخصومات:** معالجة كاملة للمرتجعات وربطها بالمخزون والمحاسبة، مع سياسات خصم مرنة.

### 💰 2. النظام المحاسبي الموحد (Unified Accounting)
*   **دليل الحسابات (COA):** هيكلية حسابات شجرية مرنة (SAP-style).
*   **الترحيل اللحظي (Auto-Journaling):** تسجيل القيود المحاسبية تلقائياً مع كل حركة مالية.
*   **التقارير المالية الاحترافية:** (ميزان المراجعة، الأرباح والخسائر، الميزانية العمومية).
*   **إدارة الضريبة:** موديول خاص لضريبة القيمة المضافة (VAT) يسهل عملية الإقرارات.

### ⚙️ 3. إدارة العملات وأسعار الصرف (Currency & Exchange Rates)
*   **إدارة العملات (OCRN):** تعريف العملات العالمية ورموزها.
*   **سجل أسعار الصرف (ORTT):** تتبع يومي لتغيرات أسعار الصرف مقابل العملة الأساسية.
*   **نظام العملة الأساسية:** تحويل آلي في كافة موديولات النظام.

### 📦 4. إدارة المخازن (Inventory Management)
*   **تتبع المخزون:** تحديث الكميات لحظياً مع حركات البيع والشراء.
*   **تنبيهات الحد الأدنى:** نظام ذكي لتنبيه الموظفين بضرورة إعادة الطلب.
*   **التصنيف الديناميكي:** إدارة الأصناف حسب الفئات والمستودعات.

### 👥 5. الموارد البشرية والرواتب (HR & Payroll)
*   **سجل الموظفين:** قاعدة بيانات متكاملة تشمل العقود والبيانات الشخصية.
*   **مسير الرواتب الآلي:** إصدار الرواتب وربطها آلياً بالحسابات البنكية للشركة والقيود المحاسبية.

## 🛠️ التقنيات المستخدمة (Tech Stack)

*   **Frontend:** [Angular 17](https://angular.io/) (RxJS, Core Components).
*   **Styling:** [Bootstrap 5](https://getbootstrap.com/) + Custom SCSS (Premium UI/UX).
*   **Icons:** FontAwesome 6.
*   **Charts:** Chart.js (لتحليلات البيانات).
*   **Storage:** LocalStorage Sync (للعمل أوفلاين).

## 🚀 كيفية التشغيل (Getting Started)

1.  **تثبيت المكتبات:**
    ```bash
    npm install
    ```
2.  **تشغيل التطبيق:**
    ```bash
    npm start
    ```
3.  **فتح الرابط في المتصفح:**
